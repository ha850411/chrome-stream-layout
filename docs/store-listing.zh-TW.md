# Chrome Web Store 商店介紹（繁體中文）

## 名稱

Chrome Stream Layout

## 簡短說明

在同一個 Chrome 分頁中排列 2、3 或 4 個直播與影片來源，支援拖曳調整窗格、全螢幕與快速重載。

## 詳細說明

Chrome Stream Layout 讓你在同一個完整視窗中，同時觀看 2、3 或 4 個直播與影片來源。貼上來源網址、選擇版面並按下套用，即可建立專屬的多畫面觀看空間。

主要功能：

- 2、3、4 畫面版面，可依當下需求快速切換
- 拖曳分隔線，自由調整每個窗格的顯示比例
- 全視窗介面，減少不必要的控制元件干擾
- 一鍵重載全部來源，快速同步更新畫面
- 一鍵切換整個版面的瀏覽器全螢幕模式
- 自動儲存來源、版面與窗格比例，下次開啟可繼續使用
- 可在設定面板切換繁體中文或英文介面，並自動記住語言偏好
- 支援一般 HTTP/HTTPS 網頁與常見直接媒體網址；部分服務會使用專屬處理方式改善嵌入體驗

使用方式：

1. 點擊擴充功能圖示，開啟或切換至版面分頁。
2. 選擇 2、3 或 4 個窗格。
3. 視需要選擇繁體中文或英文介面。
4. 將完整網址貼到各個來源欄位。
5. 按下「套用」，開始同時觀看。
6. 視需要拖曳分隔線、重載全部來源或切換全螢幕。

相容性說明：

來源網站仍可能因 DRM、登入狀態、Cloudflare 驗證、伺服器端限制、瀏覽器不支援的媒體格式，或網站主動拒絕被嵌入而無法播放。擴充功能不會繞過付費牆、帳號權限或數位內容保護機制。
