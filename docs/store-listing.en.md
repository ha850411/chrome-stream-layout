# Chrome Web Store listing (English)

## Name

Chrome Stream Layout

## Summary

Arrange 2, 3, or 4 stream and video sources in one Chrome tab with resizable panes, fullscreen viewing, and quick reload.

## Detailed description

Chrome Stream Layout lets you watch 2, 3, or 4 stream and video sources together in one full-window tab. Paste your source URLs, choose a layout, and apply it to create a focused multi-view workspace.

Key features:

- Switch between 2-, 3-, and 4-pane layouts
- Drag the dividers to resize each pane
- Use a clean, full-window interface with minimal visual obstruction
- Reload every source with one click
- Put the complete layout into browser fullscreen
- Automatically remember sources, layout, and pane sizes
- Switch the dashboard interface between Traditional Chinese and English
- Open standard HTTP/HTTPS pages and common direct media URLs, with tailored handling for selected services

How to use:

1. Click the extension icon to open or focus the layout tab.
2. Choose a 2-, 3-, or 4-pane layout.
3. Select Traditional Chinese or English for the dashboard interface if needed.
4. Paste a complete URL into each source field.
5. Select Apply to start watching.
6. Drag the dividers, reload all sources, or enter fullscreen as needed.

Compatibility note:

Some sources may not play because of DRM, sign-in requirements, Cloudflare challenges, server-side restrictions, unsupported media formats, or site logic that refuses embedding. The extension does not bypass paywalls, account permissions, or digital content protection.
