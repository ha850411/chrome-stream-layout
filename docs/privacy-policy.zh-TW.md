# Chrome Stream Layout 隱私權政策

生效日期：2026 年 7 月 14 日

Chrome Stream Layout（以下稱「本擴充功能」）重視您的隱私。本政策說明本擴充功能會處理哪些資料、處理目的，以及資料如何保存與分享。

## 資料蒐集

本擴充功能不會蒐集、販售或傳送可識別您身分的個人資料，也不使用廣告、追蹤器或分析服務。

本擴充功能會在您的裝置上儲存使用本功能所需的設定，例如：

- 您輸入的直播或影片來源網址
- 窗格數量、排列方式及窗格大小
- 介面與播放相關偏好

這些設定使用 Chrome 提供的本機儲存空間保存，不會由本擴充功能傳送至開發者或開發者控制的伺服器。您可以透過清除擴充功能資料或解除安裝本擴充功能刪除這些資料。

## 網站與第三方服務

當您將直播或影片網址加入版面時，本擴充功能會直接在 Chrome 中載入該網站的內容。該網站可能接收一般網頁請求所包含的資訊，例如 IP 位址、瀏覽器資訊、Cookie 或登入狀態，並依其自身的隱私權政策處理資料。本擴充功能無法控制第三方網站的資料處理方式。

若您加入 Bilibili 直播網址，本擴充功能會將該網址中的直播房間編號傳送至 Bilibili 官方 API，以取得實際房間編號並建立播放器網址。此請求不附帶登入憑證，但仍受 Bilibili 的服務條款及隱私權政策約束。

本擴充功能不會繞過付費牆、帳號權限、數位版權管理（DRM）或其他內容保護機制。

## 權限用途

本擴充功能使用下列 Chrome 權限：

- `storage`：在您的裝置上儲存並恢復版面、來源網址及介面設定。
- `tabs`：建立、尋找及切換至本擴充功能自己的版面分頁，避免重複開啟相同頁面。
- `declarativeNetRequest` 與 `declarativeNetRequestWithHostAccess`：僅對本擴充功能版面分頁中的子框架請求套用暫時性規則，使使用者指定的來源能以多窗格方式顯示。
- 網站存取權限：載入您指定的 HTTP 或 HTTPS 直播及影片來源，並在部分支援網站的嵌入框架中提供播放與版面輔助功能。

本擴充功能不會利用上述權限蒐集您的瀏覽紀錄，也不會將一般分頁的網頁內容傳送給開發者。

## 資料分享

除您選擇載入的第三方網站及上述 Bilibili API 請求外，本擴充功能不會將資料出售、出租或分享給開發者、廣告商、資料經紀商或其他第三方。

## 資料安全與保存

您的版面設定由 Chrome 儲存在您的裝置上，保存期間取決於您的 Chrome 設定及您是否清除擴充功能資料或解除安裝本擴充功能。雖然我們會合理維護本擴充功能的安全性，但任何軟體或網路傳輸方式均無法保證絕對安全。

## 兒童隱私

本擴充功能並非以兒童為主要使用對象，且不會蓄意蒐集兒童的個人資料。

## 政策變更

本政策可能因功能、法規或資料處理方式變更而更新。更新時將修改本頁所列的生效日期。重大變更將透過適當方式說明。
